ExinOne now supports AutoInvest. You can set up and run AutoInvest plans through the ExinOne [bot](https://mixin.one/codes/310e1f2b-2fc7-4ae0-ae2c-2f4c189808b1).


## Set up an AutoInvest plan

On the "AutoInvest" page of ExinOne, you can freely create AutoInvest plans for a single token or multiple tokens and build your own strategy.

First, follow the in-app guide to configure your plan:

![autoinvest](/assets/images/autoinvest.png)


## Deposit funds for your plan

Once your plan is set up, to make sure it runs smoothly, please deposit funds in advance. As shown below, deposit your USDT into Savings — once there is enough balance in Savings, the plan will execute automatically.

![deposit2savings](/assets/images/deposit2savings.png)

As shown above, Savings supports multiple versions of USDT. You can pick the version that matches the USDT you hold in your Mixin wallet and deposit it into Savings.


## How to get USDT

Please purchase USDT through the TIGA-Safe (@7000104225) bot. After a successful purchase, the USDT will be automatically transferred to your new-system Mixin wallet and is ready to be used for AutoInvest.

Because ExinOne currently supports AutoInvest with multiple versions of USDT, you can choose whichever USDT version has the best price, deposit it into Savings, and let the plan run automatically.
