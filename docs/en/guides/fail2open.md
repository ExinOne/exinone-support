---
title: What should I do if I can't open ExinOne?
layout: article
---

If you cannot open the ExinOne page normally, please follow the steps below to troubleshoot:

### Update Mixin to the latest version

On the Mixin chat page, search for "7000" as shown below, and send "download" to this bot. It will trigger an auto-reply with the keyword — please follow the instructions to download and install the latest version of Mixin.

![alt text](/assets/images/dldmixin.png)

### Re-authorize

On the ExinOne page that fails to open, there are three dots in the top-left corner, as shown below:

![alt text](/assets/images/fail2open.png)

Tap [View Authorizations], cancel all authorizations granted to ExinOne, then close ExinOne and reopen it to see whether it loads normally.

### Switch network

If you see the message shown in the image below, please switch your network according to the error message.

![alt text](/assets/images/3ways.png)
