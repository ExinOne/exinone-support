# ExinOne 定价规则

ExinOne 采用高效的 One-Step 服务模式，直接对接全网深度与流动性最佳的交易所 API，确保您的交易快速成交。

## 手续费明细

### 闪兑交易手续费
- **稳定币版本兑换**: 0.05% 手续费 ✅ 支持 EPC 抵扣
- **USDC/USDT**: 0.1% 手续费✅ 支持 EPC 抵扣
- **卖出 BOX**: 1% 手续费，❌ 不支持 EPC 抵扣
- **其它代币闪兑**:  0.2% 手续费，✅ 支持 EPC 抵扣


### 币币&挂单交易
- **标准费率**: 0.2%
- **EPC 抵扣**: ✅ 支持
- **市价单/限价单**: 费率一致，均为 0.2%
- **示例**: ETH/USDT 交易，手续费为交易金额的 0.2%

### BOX 交易规则
- **闪兑买入 BOX**: 0.2% 手续费，✅ 支持 EPC 抵扣
- **闪兑卖出 BOX**: 1% 手续费，❌ 不支持 EPC 抵扣
  - 注：卖出费率根据市场情况在 0.2%-1% 之间调整
- **币币交易&挂单 BOX**: 统一 0.2% 手续费，✅ 支持 EPC 抵扣


## 多步交易规则
- **收费原则**: 仅收取一次手续费，以最高费率为准
- **EPC 抵扣规则**: 如有任一步骤不支持 EPC 抵扣，则整个路径不支持抵扣
- **示例**: BOX → USDT → USDC 路径
  - BOX → USDT: 1% 手续费，不支持 EPC 抵扣
  - USDT → USDC: 0.1% 手续费，支持 EPC 抵扣
  - 最终收费: 1% 手续费，不支持 EPC 抵扣

## 其他服务
- **定投功能**: 功能本身免费，交易通过闪兑完成，按闪兑规则收费，也就是说通过定投买入代币，和直接自己去闪兑交易买入代币，结果一致，定投只是帮您完成了这个过程，不额外收取费用。

## EPC 手续费点卡

- **兑换比例**: 100 EPC = $1 手续费抵扣额
- **使用方式**: 需充值至 ExinOne EPC 账户后方可使用（位于首页左上角头像的 EPC 账户），如果 EPC 数量不足以抵扣交易手续费，则无法使用 EPC 抵扣交易费用
- **详细介绍**: 查看 [EPC 介绍](/features/epc)
